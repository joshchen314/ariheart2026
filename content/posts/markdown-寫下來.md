LyX 是相當不錯的平台，可以直接記錄想法與筆記，把排版的事交給平台處理。現在，Markdown 語法更美妙了，調出記錄與排版的黃金比例。

使用 Word 來記錄，會遇到矛盾的自我對話，到底是排版還是記錄。若只有純記錄卻完全不排版，那我還需要用這麼 FANCY 的軟體嗎？所以，純記錄的話，只要使用 MarkdownPad 2 和三個 Markdown 語法，就可以達成我的目標了。

1. 常用語法\

\
- 分段標題:\

  ```codebox
  #標題一\

  ##標題二\

  ###標題三\

  ####標題四
  ```

  \
\
- 有編號的清單 : 雖然編號是 1 ，但是，Markdown 會很貼心的改成連續編號。\

  ```codebox
  1. A\

    1. apple\

    1. apron\

    1. astronaut\

  1. B\

    1. bank\

    1. bark
  ```

  \
\
- 粗體\

  ```codebox
  **Important**
  ```

  \
\

\
2. 編輯器\

\
- MarkdownPad 2 : for Windows
\
- Nebulous Notes : for iOS
\

\
