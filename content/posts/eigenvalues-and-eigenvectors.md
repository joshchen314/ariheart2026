給定一個向量空間![\mathbf{E}](http://upload.wikimedia.org/math/5/e/b/5eb237ccb8c2716d347ab313cad7918e.png)，從![\mathbf{E}](http://upload.wikimedia.org/math/5/e/b/5eb237ccb8c2716d347ab313cad7918e.png)到![\mathbf{E}](http://upload.wikimedia.org/math/5/e/b/5eb237ccb8c2716d347ab313cad7918e.png)自身的線性變換![\mathbf{T}](http://upload.wikimedia.org/math/5/3/7/5371fbc9a1a509110366ce115ba86848.png)是—個保持向量加法和純量乘法的函數，例如旋轉、反射、拉伸、壓縮，或者這些變換的組合等等。一個線性變換可以通過它們在向量上的作用來可視化。一般來說，一個向量在經過映射之後可以變為任何可能的向量，而特徵向量具有更好的性質。
一個**線性變換 ![\mathbf{T} : \mathbf{E} \rightarrow \mathbf{E}](http://upload.wikimedia.org/math/6/b/7/6b763b47009bc19aa6ec4b4f2d5fed83.png) 的特徵向量****![v](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png)**是在這個線性變換下簡單地乘以一個純量![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png)的非零向量。也就是說![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png) 滿足：
![\mathbf{T} ( v ) = \lambda v](http://upload.wikimedia.org/math/2/9/7/297a8f679cd83b00801829146408fe04.png)其中的縮放因子稱為這個特徵向量的**特徵值**，或者說是線性變換![\mathbf{T}](http://upload.wikimedia.org/math/5/3/7/5371fbc9a1a509110366ce115ba86848.png)的特徵值。反過來，一個實數![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png)是線性變換![\mathbf{T}](http://upload.wikimedia.org/math/5/3/7/5371fbc9a1a509110366ce115ba86848.png)的一個特徵值，若且唯若有一個非零向量![v](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png)滿足上面的式子。
所有具有相同的特徵值![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png)的特徵向量和零向量一起，組成了一個向量空間，稱為線性變換的一個**特徵空間**，一般記作![\mathbb{E}_{\lambda}(\mathbf{T})](http://upload.wikimedia.org/math/f/3/1/f318d321e7b0f08e27dae2f902a139ac.png)。這個特徵空間如果是有限維的，那麼它的維數叫做![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png) 的**幾何重數**。
變換的**主特徵向量**是模最大的特徵值對應的特徵向量。有限維向量空間上的一個變換的**譜**是其所有特徵值的集合。
特徵向量也可以看作是關於係數![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png)的方程：
![\mathbf{T} ( x ) = \lambda x](http://upload.wikimedia.org/math/b/d/4/bd4703c21a14c04e805ccdc02612da1a.png)的非零解。顯然只有在![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png)是變換![\mathbf{T}](http://upload.wikimedia.org/math/5/3/7/5371fbc9a1a509110366ce115ba86848.png)的特徵值之時，方程才有非零解。
 
The terms **characteristic vector**, **characteristic value**, and **characteristic space** are also used for these concepts. The prefix **eigen-** is adopted from the German word *eigen* for "self-" or "unique to", "peculiar to", or "belonging to"

### General definition

The concept of eigenvectors and eigenvalues extends naturally to abstract linear transformations on abstract vector spaces. Namely, let ![V](http://upload.wikimedia.org/math/5/2/0/5206560a306a2e085a437fd258eb57ce.png) be any vector space over some field ![K](http://upload.wikimedia.org/math/a/5/f/a5f3c6a11b03839d46af9fb43c97c188.png) of scalars, and let ![T](http://upload.wikimedia.org/math/b/9/e/b9ece18c950afbfa6b0fdbfa4ff731d3.png) be a linear transformation mapping ![V](http://upload.wikimedia.org/math/5/2/0/5206560a306a2e085a437fd258eb57ce.png) into ![V](http://upload.wikimedia.org/math/5/2/0/5206560a306a2e085a437fd258eb57ce.png). We say that a non-zero vector ![v](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png) of ![V](http://upload.wikimedia.org/math/5/2/0/5206560a306a2e085a437fd258eb57ce.png) is an **eigenvector** of ![T](http://upload.wikimedia.org/math/b/9/e/b9ece18c950afbfa6b0fdbfa4ff731d3.png) if (and only if) there is a scalar ![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png) in ![K](http://upload.wikimedia.org/math/a/5/f/a5f3c6a11b03839d46af9fb43c97c188.png) such that

:   ![T(v)=\lambda v](http://upload.wikimedia.org/math/a/c/9/ac9e0c4c0021e610a67bfb20f44b2b76.png).

This equation is called the eigenvalue equation for ![T](http://upload.wikimedia.org/math/b/9/e/b9ece18c950afbfa6b0fdbfa4ff731d3.png), and the scalar ![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png) is the **eigenvalue** of ![T](http://upload.wikimedia.org/math/b/9/e/b9ece18c950afbfa6b0fdbfa4ff731d3.png) corresponding to the eigenvector ![v](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png). Note that ![T(v)](http://upload.wikimedia.org/math/0/8/2/08280de5642c348a3099e37d1398975d.png) means the result of applying the operator ![T](http://upload.wikimedia.org/math/b/9/e/b9ece18c950afbfa6b0fdbfa4ff731d3.png) to the vector ![v](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png), while ![\lambda v ](http://upload.wikimedia.org/math/7/c/5/7c5cfe5d8c81139c5fd23d7c9228a437.png) means the product of the scalar ![\lambda](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png) by ![v](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png).
The matrix-specific definition is a special case of this abstract definition. Namely, the vector space ![V](http://upload.wikimedia.org/math/5/2/0/5206560a306a2e085a437fd258eb57ce.png) is the set of all column vectors of a certain size ![n](http://upload.wikimedia.org/math/7/b/8/7b8b965ad4bca0e41ab51de7b31363a1.png)×1, and ![T](http://upload.wikimedia.org/math/b/9/e/b9ece18c950afbfa6b0fdbfa4ff731d3.png) is the linear transformation that consists in multiplying a vector by the given ![n\times n](http://upload.wikimedia.org/math/6/0/7/607acaa73c762411b20745149a11e90b.png) matrix ![A](http://upload.wikimedia.org/math/7/f/c/7fc56270e7a70fa81a5935b72eacbe29.png).
Some authors allow ![v](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png) to be the zero vector in the definition of eigenvector. This is reasonable as long as we define eigenvalues and eigenvectors carefully: If we would like the zero vector to be an eigenvector, then we must first define an eigenvalue of ![ T ](http://upload.wikimedia.org/math/b/9/e/b9ece18c950afbfa6b0fdbfa4ff731d3.png) as a scalar ![ \lambda ](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png) in ![K](http://upload.wikimedia.org/math/a/5/f/a5f3c6a11b03839d46af9fb43c97c188.png) such that there is a *nonzero* vector ![ v ](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png) in ![V](http://upload.wikimedia.org/math/5/2/0/5206560a306a2e085a437fd258eb57ce.png)with ![ T(v) = \lambda v ](http://upload.wikimedia.org/math/a/c/9/ac9e0c4c0021e610a67bfb20f44b2b76.png). We then define an eigenvector to be a vector ![ v ](http://upload.wikimedia.org/math/9/e/3/9e3669d19b675bd57058fd4664205d2a.png) in ![V](http://upload.wikimedia.org/math/5/2/0/5206560a306a2e085a437fd258eb57ce.png) such that there is an eigenvalue ![ \lambda ](http://upload.wikimedia.org/math/e/0/5/e05a30d96800384dd38b22851322a6b5.png) in ![K](http://upload.wikimedia.org/math/a/5/f/a5f3c6a11b03839d46af9fb43c97c188.png) with ![ T(v) = \lambda v ](http://upload.wikimedia.org/math/a/c/9/ac9e0c4c0021e610a67bfb20f44b2b76.png). This way, we ensure that it is not the case that every scalar is an eigenvalue corresponding to the zero vector.
