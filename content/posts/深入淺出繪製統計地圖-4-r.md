既然我們已經學習完以SAS繪製美麗的統計地圖，那麼接下來，我們就來試一試統計學界中最為強大的系統，R語言。同樣地，我們需要一個圖資檔，以進行繪製臺灣地圖，然後，再將圖資檔與統計資料檔進行整合後，就可以得到一張美麗的統計地圖。如同SAS繪圖的邏輯，使用R的套件是 ggplot2，並且，步驟是4步成圖。

**第一步**，我們先匯入已經轉成 big5 格式的 shp 圖資檔，資料包含到縣市的鄉鎮市區級資料。

```
library(ggplot2)
library(maptools)
library(rgeos)
library(scales)
library(ggmap)
library(RColorBrewer)
## set the shapefile
setwd("＊＊＊/RStudio/201705 TWMAP/Shapefiles/TWN_adm/2/")
states.shp <- readShapeSpatial("COUNTY20161214.shp")
class(states.shp)
```

**第二步**就是匯入統計資料檔，這個步驟獲得的統計資料檔是 agrcensus.CSV。

```
## input data to plot on map
mydata <- read.csv("＊＊＊/RStudio/201705 TWMAP/agrcensus.CSV")
head(mydata)
states.shp.f <- fortify(states.shp, region = "COUNTYCODE")
class(states.shp.f)
head(states.shp.f)
```

**第三步**就是把前二者的資料檔整合在一起。

```
#merge with coefficients and reorder
merge.shp.coef <- merge(states.shp.f, mydata, by="id", all.x=TRUE)
final.plot <- merge.shp.coef[order(merge.shp.coef$order), ]
```

**第四步**，以 ggplot2 套件繪圖。需要修改的參數為 「fill」欄位的統計量，這裡的例子是 「FHN」；「color」欄位是指地理界線的顏色，這裡是縣市的顏色；「geom\_text」欄位是指縣市名稱標籤的屬性。

```
#basic plot
ggplot() +
 geom_polygon(data = final.plot,
 aes(x = long, y = lat, group = group,
     fill = FHN),
     color = "black", size = 0.25) +
 coord_map()
```

```
#plot using geom_text: *****
##aggregate data to get mean latitude and mean longitude for each state
cnames <- aggregate(cbind(long, lat) ~ COUNTYNAME, data=final.plot, FUN=function(x) mean(range(x)))
ggplot() +
 geom_polygon(data = final.plot,
 aes(x = long, y = lat, group = group, fill = FHN),
 color = "black", size = 0.25) +
 coord_map() +
 scale_fill_distiller(name="Enterprise", palette = "YlGn", breaks = pretty_breaks(n = 5))+
 theme_nothing(legend = TRUE)+
 labs(title="FHN of AGR in Taiwan")+
 geom_text(data=cnames, aes(long, lat, label = COUNTYNAME), size=3, fontface="bold")
```
